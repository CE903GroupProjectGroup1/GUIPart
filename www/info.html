Manual

You can find some command to stop or start some part of the application.
On cseepimanX machines
  • In order to serve monitoring index.html page on the port 8000(Can be changed to allowed ports) a python server needs to be run. Please run command below when you are in the directory of the index.html.
    nohup python3 -m http.server 8000 &
  • In order to listen and transform the data coming from brambles two python files need to be run. Please run commands below when you are in the directory of the index.html and make sure DataTransformer.py and UDPListener.py is in the directory.
    nohup python DataTransformer.py &
    nohup python UDPListener.py &
