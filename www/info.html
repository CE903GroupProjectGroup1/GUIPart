Manual

You can find some command to stop or start some part of the application.
On cseepimanX machines
  • In order to serve monitoring index.html page on the port 8000(Can be changed to allowed ports see startCM.sh) a python server needs to be run. Please run command below when you are in the directory of the index.html or www.
    bash startCM.sh
On cseepimanX machines
  • In order to stop serving the index.html page the command below needs to be run. This command will close all python proccess that user run.
    bash stopCM.sh
On pinodexxx rasperypi machines
  • Copy CPUMemory_UDPSender.c file onto a directory on a Raspberry node.
    scp CPUMemory_UDPSender YourID@pinode001:/home/YourID

    Login to the node and compile the program.

    gcc CPUMemory_UDPSender.c -o CPUMemory_UDPSender

    Distribute the compiled program to other nodes.
    scp CPUMemory_UDPSender YourID@pinode002

    Running the agent program
    Login to each pinode then run the following command
    ./CPUMemory_UDPSender &

    Alternatively run the following script from CSEEPIMAN2 for each node.
    ssh pinode001 ‘nohup ./CPUMemory_UDPSender’ &
    ssh pinode002 ‘nohup ./CPUMemory_UDPSender’ &

    Stopping all the agents by a single script is something we could not complete.
    At the moment, just closing ‘StartPiAgents.sh’ stops all the agents.
    close ‘StartPiAgents.sh’ terminal window.
